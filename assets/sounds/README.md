# Screamer Sounds

Добавь сюда звуки.

## Нужно:
- `scream.mp3` — основной звук скримера (0.5-1 сек)
- `ambient.mp3` — фоновый звук (опционально)

## Источники бесплатных звуков:
- freesound.org
- mixkit.co
- pixabay.com/sound-effects

## Для тестовой версии:
Используется синтезированный звук через Web Audio API.

## Для релиза:
Добавить реальные страшные звуки.
