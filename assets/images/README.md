# Screamer Images

Добавь сюда страшные картинки для скримеров.

## Рекомендации:
- Формат: PNG или WebP
- Размер: 500x500 px минимум
- Оптимизация: сжать для быстрой загрузки

## Для тестовой версии:
- 1 картинка (creepy но не жесть)

## Для релиза:
- 3-4 разных варианта
- Разная степень страшности
